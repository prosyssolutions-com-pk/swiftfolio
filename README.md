# Swiftfolio

Swiftfolio is a modern responsive portfolio template for web designers and developers.

## Features

- Responsive layout
- Portfolio grid
- Scroll animations
- Mobile navigation
- Clean UI design

## Technologies

HTML5  
CSS3  
JavaScript
